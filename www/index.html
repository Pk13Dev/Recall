    Library overview

    Guide

    How RECALL works

    RECALL keeps the home screen focused on studying. This page explains the flow, the buttons, and where your quizzes are saved.

    1

    Import

    Add one quiz, import a whole folder, or launch the demo.

    2

    Organize

    Use folders, overview quizzes, rename, move, and delete tools.

    3

    Study

    Answer questions, finish a run, and review your analytics.

    Top Bar

    Global controls

    • Theme

      Change the visual style

      Opens the theme picker and remembers your choice for later sessions.

    • Sound

      Adjust quiz sounds

      Changes the notification volume for correct, wrong, win, and loss effects.

    • Analytics

      Open your learning history

      Shows recent scores, quiz performance, most-missed questions, recent answers, and total study time.

    • Guide

      Return to this walkthrough

      Keeps the main screen uncluttered by moving the longer explanations here.

    Import

    Getting quizzes in

    • Upload JSON

      Import one quiz file

      Opens a file picker for a single JSON quiz and starts that quiz after import.

    • Open Folder

      Import many quizzes at once

      Brings in every supported JSON file in a selected folder and recreates nested folders in Saved Quizzes.

    • Drop Zone

      Drag files straight into RECALL

      You can drop quiz files or folders onto the upload panel instead of using the file picker buttons.

    • Use Demo Quiz

      Preview the experience

      Launches a built-in sample quiz without saving a new file to your library first.

    Library

    Saved Quizzes buttons

    • New Folder

      Create a folder in the current location

      Adds a new folder inside the folder you are currently viewing.

    • Overview

      Build a mixed review quiz

      Creates an overview quiz by pulling one random question from each regular quiz in the current folder.

    • Up

      Go to the parent folder

      Moves one level higher in the Saved Quizzes folder tree.

    • Breadcrumbs

      Jump to any folder in the path

      Use the path buttons above the list to move directly to a previous folder level.

    Items

    Folder and quiz actions

    • Open

      Enter a folder

      Shows the quizzes and subfolders stored inside that folder.

    • Load

      Start a saved quiz

      Loads the selected quiz into the quiz screen and begins a fresh attempt.

    • Rename

      Edit the visible name

      Updates the label of a folder or quiz while keeping it in the library.

    • Move

      Send a quiz to another folder

      Opens the inline picker so you can choose a new folder destination for that quiz.

    • Delete

      Remove a folder or quiz

      Deletes the selected item. Folder removal can also move its contents up instead of deleting everything inside.

    Quiz Flow

    What happens during a quiz

    • Answer

      Pick one option per question

      RECALL locks the answers, marks the result, and updates your score immediately after you choose.

    • Next Question

      Continue only after answering

      The button stays disabled until you select an answer for the current question.

    • Quiz Length

      Sampled study runs

      Each run can sample up to 20 questions, and each question shows 4 answer choices.

    • Results

      Review and continue

      Use Restart to return to the library or View Analytics to inspect your study history.

    Storage

    Where your quizzes are saved

    Browser local

    RECALL keeps quizzes, folders, theme, sound, and analytics in this browser.

    • Browser storage is the default and keeps quizzes, folders, theme, sound, and analytics in this browser.
    • Saved quizzes stay available in this browser on this device when browser storage is supported.
    • Temporary mode can still work in restricted environments, but those saves only last for the current tab session.

    Remove Folder

    If unchecked, the folder is removed and its contents move into the parent directory.

    Remove Quiz

    This removes the saved quiz from RECALL.
